# War Thunder 研发计算器 v1.0.0

## 下载选择

- `WarThunderResearchCalculator-v1.0.0-portable.zip`
  - 推荐普通玩家下载。
  - 内置 `node.exe` 和 `node_modules`。
  - 解压后双击 `WarThunderResearchCalculator.exe` 即可使用。

- `WarThunderResearchCalculator-v1.0.0.zip`
  - 轻量版，适合已经安装 Node.js 的用户。
  - 解压后先运行 `npm install`，再双击 `WarThunderResearchCalculator.exe` 或运行 `npm start`。

## 功能亮点

- 支持多国家科技树浏览。
- 支持陆战、空战、直升机、远洋舰队、近岸舰队。
- 自动计算所需 RP 和银狮。
- 支持“只算所选 / 包含前置”两种计算方式。
- 正确处理叠加载具，不会错误地把整组都算作前置。
- Rank I 初始载具会自动标记为已解锁。
- 研发计划会按等级和科技树顺序自动排列。
